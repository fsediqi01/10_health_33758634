# Health & Fitness Tracker

A Node.js + Express + EJS + MySQL fitness tracking app for the Web Applications coursework.

## Features
- Add fitness activities
- View all activities
- Search activities in database
- About page
- Fully working MySQL integration
- Deployable on Goldsmiths VM

## Installation
npm install

## Run
node index.js (listens on port 8000)

## Database
create_db.sql  
insert_test_data.sql  

## Deployment
Configured for Goldsmiths student VM.
